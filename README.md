# Seguimiento de Personas Nuevas

Aplicación web desarrollada para el Movimiento Misionero Sin Fronteras.

## Funcionalidades

- Inicio de sesión mediante código.
- Listado de personas.
- Consulta de información.
- Registro de seguimientos.
- Historial de seguimientos.
- Control por sedes.
- Control por roles.
- Integración con Google Apps Script y Google Sheets.

## Tecnologías

- HTML
- CSS
- JavaScript
- Google Apps Script
- Google Sheets
